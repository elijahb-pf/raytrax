# Limitations

The following is a list of current limitations of Raytrax.

## Input formats

- To load magnetic equilibria, only [VMEC++](https://proximafusion.github.io/vmecpp/) is supported so far. Note that VMEC++ is interoperable with SIMSOPT and can read legacy VMEC output files as well.


## Physics

- For tracing, only the cold plasma dielectric tensor is implemented so far.

## Validation

- Raytrax has not been fully validated against existing ray traxing codes or experimental data yet.
